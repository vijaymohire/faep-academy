# Quantum Computing Case Studies

This collection provides curated case studies demonstrating the practical application of Quantum Computing across optimization, machine learning, cryptography, chemistry, materials science, financial modeling, logistics, and scientific computing.

These case studies complement the FAEP Academy courses, tutorials, laboratories, notebooks, and engineering projects.

---

# Objectives

This collection helps learners:

- Understand practical Quantum Computing applications
- Study quantum engineering architectures
- Learn engineering approaches to hybrid quantum solutions
- Analyze implementation challenges and limitations
- Evaluate quantum advantage for real-world problems
- Explore emerging quantum technologies
