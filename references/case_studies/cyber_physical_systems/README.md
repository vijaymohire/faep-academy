# Cyber-Physical Systems Case Studies

This collection provides curated case studies demonstrating the practical application of Cyber-Physical Systems (CPS) across smart manufacturing, industrial automation, robotics, healthcare, transportation, energy systems, aerospace, and intelligent infrastructure.

These case studies complement the FAEP Academy courses, tutorials, laboratories, notebooks, and engineering projects.

---

# Objectives

This collection helps learners:

- Understand Cyber-Physical Systems architectures
- Study integration of physical and digital systems
- Learn engineering approaches to CPS design and deployment
- Analyze real-world implementation challenges
- Evaluate system safety, reliability, and security
- Explore emerging CPS technologies
