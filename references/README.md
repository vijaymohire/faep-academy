# FAEP Reference Library

The **FAEP Reference Library** provides a centralized collection of books, research papers, online resources, course material, and technical references that support learning across all FAEP Academy courses.

Unlike course-specific learning material, this repository contains reusable reference resources shared by multiple engineering disciplines.
