# 1. Purpose

This document defines the Career Path taxonomy used throughout the FAEP Academy.

Career Paths organize learning journeys that prepare learners for progressive professional roles by combining knowledge, skills, competencies, certifications, practical experience, and continuous learning.

They help learners understand how educational achievements translate into real-world career progression.

---

# 2. Design Principles

Career Paths should be:

- Learner-centric
- Competency-driven
- Industry-aligned
- Flexible
- Progressive
- Reusable
- Knowledge Graph ready
