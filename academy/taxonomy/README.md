# Overview

The **FAEP Academy Taxonomy** defines the hierarchical organization and classification of educational knowledge throughout the Academy.

It provides a structured vocabulary for organizing learning assets, competencies, technologies, domains, and professional pathways.

The taxonomy enables consistent navigation, intelligent discovery, metadata standardization, Knowledge Graph construction, and AI-powered learning recommendations.

---

# Objectives

The Taxonomy framework aims to:

- Organize educational knowledge consistently.
- Standardize domain classification.
- Support metadata and tagging.
- Enable semantic search.
- Power Knowledge Graph relationships.
- Improve repository navigation.
- Support personalized learning paths.
- Scale across emerging technologies.

# Relationship with Metadata

Every educational asset is classified using the Academy Taxonomy.

```text
Educational Asset
        │
        ▼
Metadata
        │
        ▼
Taxonomy
        │
        ▼
Knowledge Graph
```

The taxonomy provides the controlled vocabulary used by the Universal Metadata Schema.

---

# Relationship with Knowledge Graph

The taxonomy defines the classification hierarchy, while the Knowledge Graph captures the relationships between entities.

```text
Taxonomy
    │
    ▼
Metadata
    │
    ▼
Knowledge Graph
    │
    ▼
AI Reasoning
```
