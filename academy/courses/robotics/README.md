# Robotics

Develop the knowledge and engineering skills required to design, build, program, integrate, and operate intelligent robotic systems that combine mechanical engineering, electronics, embedded computing, sensing, control systems, Artificial Intelligence, and autonomous decision-making.

Robotics represents the integration of multiple engineering disciplines to create intelligent systems capable of interacting safely and effectively with the physical world.

# Course Overview

Robotics combines software engineering, embedded systems, sensing, control, Artificial Intelligence, Cyber-Physical Systems, and intelligent automation to build autonomous and semi-autonomous machines.

This course provides a structured pathway through tutorials, laboratories, notebooks, engineering projects, and the FAEP Client Lab to develop practical robotics engineering skills.

---

# Learning Objectives

After completing this course, learners should be able to:

- Understand robotics engineering principles.
- Explain robotic system architectures.
- Integrate sensors, actuators, and controllers.
- Apply Artificial Intelligence to robotic systems.
- Understand autonomous navigation and control.
- Develop robotics engineering projects.
- Build a professional robotics engineering portfolio.
