# 1. Purpose

Mega Tags define the controlled vocabulary used throughout the FAEP Academy.

They provide consistent tagging across educational assets, enabling:

- Intelligent Search
- Repository Navigation
- Knowledge Graph Construction
- AI Recommendations
- Learning Analytics
- Competency Mapping
- Metadata Validation
- Content Discovery

Every educational asset should use standardized Mega Tags.

---

# 2. Design Principles

Tags should be

- Consistent
- Reusable
- Machine-readable
- Human-readable
- Non-duplicated
- Extensible
- Hierarchical
- Knowledge Graph Ready

# 17. Tag Naming Rules

Tags should

- Use Title Case
- Avoid abbreviations where possible
- Avoid duplicate meanings
- Prefer singular nouns
- Be reusable across domains

Examples

✔ Quantum Computing

✔ Digital Twins

✔ Systems Engineering

✘ QC

✘ AI Stuff

✘ Misc
