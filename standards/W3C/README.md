# FAEP W3C Standards

## Overview

The **FAEP W3C Standards Library** provides a structured reference to web standards, semantic technologies and interoperability specifications developed by the World Wide Web Consortium (W3C).

W3C standards enable open, interoperable and accessible web technologies that support modern enterprise applications, cloud platforms, semantic knowledge systems, artificial intelligence, digital twins and distributed intelligent ecosystems.

Within the FAEP Academy, W3C standards complement the QAI Computing Foundations, FAEP Reference Architecture, Engineering Frameworks, Engineering Patterns and Technology Libraries by providing the web and semantic interoperability foundation for intelligent systems.

This is a very appropriate document to conclude the Standards Library.

W3C (World Wide Web Consortium) complements the other standards by providing the web interoperability and semantic web foundation.

Unlike IEEE or ISO, W3C is focused on ensuring that web technologies remain open, interoperable and machine-readable, which fits well with the knowledge-centric architecture you've been building.
