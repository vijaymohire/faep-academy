# Overview

The **Drones** laboratory provides practical, hands-on learning experiences for designing, developing, validating, and deploying intelligent unmanned aerial systems (UAS) within the FAEP (Federated Autonomous Ecosystem Platform).

These laboratories prepare learners to engineer intelligent aerial systems capable of collaborating with enterprise platforms, autonomous agents, robotics, Cyber-Physical Systems (CPS), and future hybrid quantum-classical ecosystems.
